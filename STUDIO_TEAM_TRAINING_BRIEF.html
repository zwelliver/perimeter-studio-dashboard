Studio Automation System - Quick Start Guide

What This System Does

Automatically handles capacity tracking, task creation, and production scheduling so you can focus on creating great content instead of managing spreadsheets.

1. The Four Asana Projects

Your workflow follows this path:

  • Forecast - Long-range planning (3-6 months out) for strategic initiatives and future content
  • Preproduction (1208336083003480) - Scout, plan, prepare
  • Production (1209597979075357) - Film, record, capture
  • Post Production (1209581743268502) - Edit, review, publish

The Forecast Project

The Forecast project is our forward-thinking planning tool. Tasks here represent future content ideas, strategic initiatives, and long-range planning (typically 3-6 months out). This allows us to:

  • Plan capacity for major initiatives before they become urgent
  • Align studio work with organizational strategic goals
  • Identify resource needs early
  • Show leadership our proactive planning

When tasks in Forecast are ready to begin active work, they move to Preproduction.

2. What YOU Do (Manual Tasks)

Task Creation

  • Create tasks in the appropriate project
  • Include video duration in title/notes: "5-minute testimonial video"
  • Set Film Date if applicable
  • Move tasks between projects as work progresses
  • Mark tasks complete when done

WOV Scheduling

  • Schedule WOV events in Google Calendar with "WOV" in the title
  • Invite the talent to the calendar event
  • That's it! The system creates the Asana task automatically

4. Understanding Capacity (% Allocation)

The system automatically calculates how much of your weekly capacity each task consumes based on:

  • Category - AI determines if it's Video, Event, Graphics, etc.
  • Duration - Extracted from your task title/notes
  • Type - AI classifies as Equipping, Enriching, Celebrational, etc.

Example

Task: "5-minute equipping video on quantum computing"

AI Analysis:

  • Category: Full Video (15% base for 5 min)
  • Type: Equipping (+30% modifier)
  • Result: 19.5% allocation (one person spends ~20% of their week on this)

5. The Dashboard - Your Mission Control

View index.html for real-time insights (updates every 15 minutes):

  • Production Timeline - What's filming this week/next week
  • Performance Metrics - Are estimates accurate?
  • Delivery Performance - On-time completion rates

6. WOV Automation

Create a Google Calendar event with "WOV" in the title

Within 30 minutes, the system automatically:

  • Creates task in Production project: "WOV - [Talent Name]"
  • Sets Film Date from calendar event
  • Calculates Due Date (Thursday 10am before Sunday it airs)
  • Sets all fields: Scheduled, Approved, WOV Set, Enriching, Communications
  • Generates 4 AI backdrop options and attaches to task

7. AI-Powered Backdrops

Every 30 minutes, the system:

  • Checks all Production tasks
  • Generates 4 professional backdrop variations using AI
  • Automatically attaches images to the Asana task
  • No manual requests needed

8. Best Practices

DO:

  • Include video duration in task titles: "8-minute testimonial"
  • Set Film Dates for filming tasks
  • Mark tasks complete (don't delete them - we need the data!)
  • Add notes about what took longer/shorter than expected
  • Use "WOV" in calendar event titles for auto-creation

DON'T:

  • Manually edit % Allocation (system recalculates every 15 min)
  • Delete tasks (mark complete instead)

10. Common Questions

Q: My task doesn't have % Allocation yet

Wait up to 15 minutes. Make sure you included duration in the title/notes.

Q: I created a WOV calendar event but no task appeared

Wait 30 minutes. Verify "WOV" is in the title and event has a non-studio attendee.

Q: Dashboard isn't updating

Check the timestamp at top. System updates every 15 min. Hard refresh your browser (Ctrl+F5 / Cmd+Shift+R).

Q: Where are the AI-generated backdrops?

Check the Asana task attachments. They're automatically added within 30 minutes of task creation.
