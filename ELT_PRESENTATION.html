1. Consistency: Standardized, Predictable Workflows

Zero Manual Data Entry

Every task is automatically analyzed, categorized, and capacity-scored within 15 minutes of creation. This eliminates human error and ensures consistent data across all projects.

Consistent Processes

  • Automated Task Classification: AI determines category and type for every task using the same criteria every time
  • Standardized Capacity Calculation: All tasks measured against the same baseline formulas
  • Uniform Calendar Integration: Film dates and due dates automatically sync to Google Calendar
  • Predictable WOV Workflow: Calendar event automatically triggers complete task creation with all fields populated

What This Means for ELT

You can trust the data. Whether you're looking at capacity today or three months ago, the same measurement standards apply. Decision-making is based on consistent, reliable metrics.

2. How the System Enforces Your Strategic Priorities

Automated Priority Scoring

Every video project request is automatically scored using AI based on three factors:

Priority Score = Purpose + Audience + Scope

Purpose (1-4 points):

  • 4 = Equipping (Teaching, Strategic Partnerships, Jeff Norris content)
  • 3 = Enriching (WOV, Testimonies, Campaign updates)
  • 2 = Ministry Support (Event promos, recaps, ministry profiles)
  • 1 = Partner Support (Church plants, nonprofits, global partners)

Audience (1-4 points): Church-wide (4), Community (3), People Groups (2), Ministry Groups (1)

Scope (1-4 points): Evergreen content (4), Seasonal/Campaign (3), Event-driven (2), One-time (1)

This scoring system ensures high-priority work (Spiritual Formation, Pastoral/Strategic) automatically gets scheduled first.

3. Staff Health: Sustainable Workloads

Preventing Burnout Through Visibility

The system tracks capacity allocation in real-time, providing early warnings when team workload exceeds sustainable levels (>100% capacity).

How We Protect Staff Health

Automatic Capacity Monitoring

  • Weekly Capacity Reports: Every Monday, team receives capacity analysis showing if we're overloaded
  • Color-Coded Warnings: Dashboard shows green (healthy), yellow (high), red (overloaded) status
  • Historical Tracking: System learns actual time tasks take and improves estimates
  • Actual vs. Estimated Variance: Shows when estimates are consistently wrong so we can adjust

Reduced Manual Overhead

Staff no longer spend time on:

  • Manual capacity spreadsheets
  • Creating WOV tasks from calendar events
  • Manually syncing film dates to calendar
  • Requesting and waiting for production backdrops
  • Updating dashboards and reports

Result: More time for creative work, less time on administrative tasks
